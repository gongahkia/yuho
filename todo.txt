### 1. CLI Help Overview


### 2. Validation with Error Messages

```bash
yuho check examples/simple_statute.yh
```

### 3. Transpilation to Mermaid Diagrams

```bash
yuho transpile examples/simple_statute.yh --target mermaid
```

### 4. Transpilation to Plain English

```bash
yuho transpile examples/simple_statute.yh --target english
```

### 5. Live Preview

```bash
yuho preview examples/simple_statute.yh
```

### 6. AST Visualization

```bash
yuho ast examples/simple_statute.yh
```

### 7. Interactive REPL

```bash
yuho repl
```

### 8. Testing with Coverage

```bash
yuho test examples/simple_statute.yh --coverage
```

### 9. Diff Between Files

```bash
yuho diff examples/simple_statute.yh examples/simple_statute.yh
```

### 10. JSON Output

```bash
yuho transpile examples/simple_statute.yh --target json
```
