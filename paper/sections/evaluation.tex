\subsection{Coverage}
\label{subsec:coverage_eval}
We encoded all \statRawSections{} sections of the Singapore Penal Code 1871.
\statLOnePass{}/\statRawSections{} ($100\%$) parse and build (L1+L2);
\statLThreePass{}/\statRawSections{} ($\sim$23\%) carry an L3 human stamp.
The unstamped long tail is dominated by short interpretation sections where
the encoding is a faithful mirror and the bug surface is small;
\S\ref{sec:limitations} discusses the cost-benefit of pushing L3 higher.

\subsection{Threats to validity}
Single jurisdiction; single statute family (penal code); the L3 reviewer
is the same author as the encoder, so signoff is partly self-marking.
Cross-jurisdiction generalisation (e.g. the Indian Penal Code, which shares
the same drafting heritage) is left to future work.
