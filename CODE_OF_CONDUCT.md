# Code of Conduct

Yuho follows the Contributor Covenant Code of Conduct.

## Expected Behavior

- Be respectful and direct.
- Keep technical disagreement focused on evidence.
- Avoid harassment, threats, personal attacks, and discriminatory language.

## Reporting

Report conduct concerns to `gabrielzmong@gmail.com`.

Reports are handled privately where possible. Maintainers may remove comments,
issues, pull requests, or access for behavior that violates this policy.
