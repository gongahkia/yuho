# Transpiler

## Transpile `.yh` to `.json`

``` py
$ python<versionNumber> trans.py
```

* Read `.yh` files are in `./web/dep/yh`.
* Transpiled `.json` files are in `./web/out/json`.
* Valid `.json` files are in `./web/dep/json`.
