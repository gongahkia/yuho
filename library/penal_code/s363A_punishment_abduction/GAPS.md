## GA

Statute text: "Whoever abducts any person shall be punished with imprisonment for a term which may extend to 7 years, or with fine, or with caning, or with any combination of such punishments."

The current Yuho grammar supports `fine := unlimited` for an uncapped fine, but `caning` requires a numeric stroke count or range. Section 363A preserves caning as a possible punishment without specifying any number of strokes, so the caning limb is retained verbatim in `supplementary` rather than fabricated as a numeric `caning := ... strokes` clause.
