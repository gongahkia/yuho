## GA

"shall also be liable to fine or to caning."

The current Yuho grammar supports `fine := unlimited` for an uncapped fine, but `caning` requires a numeric stroke count. Section 388 states liability to caning without specifying any minimum or maximum number of strokes, so the caning limb is preserved verbatim in `supplementary` rather than fabricated as a numeric `caning := ... strokes` clause.
