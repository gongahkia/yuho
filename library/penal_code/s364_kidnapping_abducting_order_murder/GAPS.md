## GA
Statute text: "shall be punished with death or imprisonment for life"

Current Yuho grammar does not provide a structured `imprisonment := life` form, so the life-imprisonment limb is preserved verbatim in `supplementary`.

## GB
Statute text: "shall, if he is not sentenced to death, also be liable to caning"

Current Yuho grammar requires a numeric caning count or range. Section 364 provides liability to caning without a stroke count, so that limb is preserved verbatim in `supplementary`.
