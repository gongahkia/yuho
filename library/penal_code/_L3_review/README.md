# L3 Review Archive

This directory contains historical Phase-D reviewer snapshots used while
moving the Singapore Penal Code library to full L3 coverage.

The active coverage state is `library/penal_code/_coverage/coverage.json`
and the active summary is `library/penal_code/_L3_flags.md`. As of the
current corpus, all 524 sections are author-stamped at L3 and there are
no active per-section `_L3_FLAG.md` files.
