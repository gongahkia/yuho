## GA
Statute text: "shall also be liable to fine or to caning. [15/2019]"

The current Yuho grammar supports `fine := unlimited`, but `caning` requires an explicit stroke count or numeric range. Section 377BI states liability to caning without specifying any number of strokes, so the caning limb is preserved in `supplementary` text rather than fabricated as a numeric `caning := ... strokes` clause.
