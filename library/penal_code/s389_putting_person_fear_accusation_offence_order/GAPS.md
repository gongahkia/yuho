## GA

Statute text: "shall also be liable to fine or to caning."

The current Yuho grammar supports `fine := unlimited`, but `caning` requires a numeric stroke count or range. Section 389 states liability to caning without specifying any minimum or maximum number of strokes, so the caning limb is preserved verbatim in `supplementary` rather than fabricated as a numeric `caning := ... strokes` clause.
